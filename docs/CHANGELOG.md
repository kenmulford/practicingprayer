## What's New (since 1.2.1)

- **On-this-day answered prayers** — Your home dashboard now surfaces prayers that were answered on this same date in prior years, so you can remember what God has already done.
- **Full screen reader support** — Complete accessibility overhaul: the home dashboard, card lists, prayer lists, tags, collections, and filter chips all read as single clear announcements. Favorite, answered, expand/collapse, and multi-select actions are now announced. A new "Select" toolbar button provides screen-reader-friendly multi-select entry. FAQ hints update dynamically.
- **Bigger splash screen icon** — The praying hands icon is now larger and sharper on the launch screen, especially on high-resolution Android devices.
