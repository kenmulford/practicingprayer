## What's New (since 1.2.1)

- **On-this-day answered prayers** — Your home dashboard now surfaces prayers that were answered on this same date in prior years, so you can remember what God has already done.
- **Better screen reader support** — Improvements throughout the home dashboard, card lists, and filter chips so TalkBack and VoiceOver read each row as a single announcement instead of fragmenting it.
