## What's New (since 1.1.0b50)

- **New tag picker** — Adding tags to a prayer now opens a dedicated tag screen where you can search, create, and remove tags in one place. Type commas to quickly add multiple tags at once (e.g., "worship, healing, family").
- **Fixed tag suggestions not responding to taps on iOS** — Tapping a suggested tag in the old inline list did nothing on iPhone/iPad. The new tag picker screen eliminates this issue entirely.
- **Improved dropdown fields** — Picker fields (card, collection, frequency, day) now show a clear outlined border with a dropdown arrow, making them easier to find and tap on all devices.
