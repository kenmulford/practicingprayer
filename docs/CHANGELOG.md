# Changelog

> App store release notes, maintained continuously. Reset when a new version/build tag is created.
>
> **Current baseline:** `1.0.4b23`

---

## What's New (since 1.0.4 build 23)

### Bug Fixes

- Fixed Prayer Time swipe getting stuck after swiping back to the first card on iOS.
- Fixed long-press to delete a custom tag color not working on either platform.
- Fixed tag color tap-to-select not responding on Android.
- Fixed tag name and color changes not refreshing across the app after saving.
- Fixed notification switch showing incorrect styling when the Settings page first loads.
- Fixed tag suggestion list being hard to see on the prayer form.
- Fixed tapping a tag suggestion not adding the tag on either platform.
- Fixed iOS keyboard covering the Save button when adding tags to a prayer.
- Fixed crash when opening a prayer request on Android.
- **Swipe to finish Prayer Time** — you can now swipe past the last prayer to reach the completion screen. Once there, you can't swipe back.

### Improvements

- **Prayer request count badge** — Each prayer card now shows the number of active requests when collapsed, so you can see at a glance how many prayers are in each card.
- **Smarter overdue list** — New prayers that haven't been prayed yet no longer appear as overdue. Only prayers you've previously prayed but haven't returned to will show up.
